Conheça o JaValet!

JaValet é um Sistema de Gerenciamento de Estacionamentos, visando a automatização de tarefas recorrentes no cotidiano desses estabelecimentos.

Sobre o Projeto

JaValet é um projeto interdisciplinar idealizado pelos alunos do 5° Semestre do Bacharelado de Ciências da Computação da IFSP do Câmpus Salto, com o intuito de aplicar os conhecimentos da linguagem Java, seus frameworks e alguns conceitos de Programação Web em um cenário prático.
